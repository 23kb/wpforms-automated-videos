WPForms Interactions — QC Index

Each interaction in videos/_shared/wpforms-interactions.js has a standalone demo on this page. Each demo loads a real WPForms snapshot, mounts a Cursor, and runs the interaction end-to-end. Click through, watch the snapshot-swap or DOM change land, then approve or red-line. See usage doc and motion-primitives QC index for the underlying primitives.
