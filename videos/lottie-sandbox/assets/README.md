# Lottie Sandbox Assets

`bumper.json` is a tiny hand-authored Lottie animation created for this repo as
CC0/public-domain demo data. It uses a single SVG-rendered shape layer with
scale and opacity keyframes over 60 frames.

`badge.json` is a hand-authored CC0/public-domain demo asset with embedded
markers (`enter`, `hold`, `exit`) for marker-driven scrubbing examples.
